cusnotes is a user-friendly open source platform created for students to easily download and share notes related to the colleges which come under cluster university srinagar or colleges following NEP 2020. Built using HTML, CSS, and JavaScript, this platform allows you to find and contribute valuable study materials, making education more accessible to everyone.
